## Getting Started

This is a program for compiling PNJava language into Hack VM code. please enter the full filepath of the folder containing your .png image files to launch.json in project11/.vscode/

"args": "Z:/ECS/Project11/Project11/src/IMGS" 

(Replace the file path with your folder of choice)

Simply run Main.java to create your output - Filename.vm 

## Included Files

Main.java - runs the program and fileIO
Symbol.java - object used for symbol table handling
PNGParser - object used for code parsing, I was definitley not trying to spell XML. 
launch.json - please enter runtime arguments here
IMGs - folder containing test programs
PNJava Jack conversion sheet.pdf - conversions from Jack syntax to PNJava pixel information
